# Build from source
### In this module, you will learn how to compile Kydra source code.

## Steps

### Requirements

- Node.js +18

### Step 1.

- First, install the dependencies with:

```bash
npm install
```

### Step 2.

- Now, compile.

```bash
npm run dist
```

### Outro.

- Then, to test in a development environment, run:

```bash
npm start
```

## Done!
